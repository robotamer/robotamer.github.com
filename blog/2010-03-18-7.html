Creating models for the Zend Framework from command line via phpcli.

Thu, Mar 18, 2010

Creating models for the Zend Framework from command line via phpcli.

php zend/model.php -h

This is a command line PHP script.

Set table

Options: -path=[ ] -table=[ ]

Usage: php zend/model.php -arg=value

~/phpcli$ php zend/model.php -path=/var/www/TaMeR/data/tmp.db3 -table=users

Sample Output:

Database Structure:
    CREATE TABLE users (
    Alias TEXT UNIQUE,
    Password TEXT,
    eMail TEXT,
    Avatar TEXT,
    Timezone TEXT,
    IP TEXT,
    count INTEGER,
    cookie TEXT,
    vars TEXT,
    status INTEGER DEFAULT 1,
    added date,
    updated datetime,
    PRIMARY KEY(Alias)
);

-

~/phpcli$ php zend/model.php -path=/var/www/TaMeR/data/tmp.db3 -table=user
