GoTamer Conv
============

####Converts from 

 * string to byte 
 * string to int 
 * string to uint
 * int    to string
 * uint   to string
 * byte   to string 
 * byte   to int
 * []byte to string 
 * string to []byte 
 * ...



 * [Pkg Documantation](http://go.pkgdoc.org/bitbucket.org/gotamer/conv "GoTamer Conversion Pkg Documentation")
 * [Repository](https://bitbucket.org/gotamer/conv "GoTamer Conversion Repository")
